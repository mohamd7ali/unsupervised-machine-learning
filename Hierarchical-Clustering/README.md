# Hierarchical Clustering

This module provides a comprehensive, hands-on exploration of Hierarchical Clustering using Python and scikit-learn. The included Jupyter Notebook demonstrates the theory, implementation, and visualization of hierarchical clustering techniques, with practical examples and clear explanations.

## Contents

- **HierarchicalClustering.ipynb**: Main notebook covering:
	- Introduction to hierarchical clustering (agglomerative and divisive)
	- Dendrogram construction and interpretation
	- Silhouette score analysis for cluster quality
	- Application to synthetic data and the Iris dataset
	- Visualizations of clustering results and dendrograms

## Features

- Step-by-step explanations of hierarchical clustering concepts
- Visual guides for dendrograms and linkage types
- Code examples for clustering, evaluation, and visualization
- Application to real-world data (Iris dataset)
- Silhouette analysis for optimal cluster selection

## Getting Started

1. **Install dependencies:**
	 - Python 3.7+
	 - Required packages: `numpy`, `pandas`, `scikit-learn`, `matplotlib`, `scipy`, `IPython`
	 - Install with pip:
		 ```bash
		 pip install numpy pandas scikit-learn matplotlib scipy ipython
		 ```
2. **Open the notebook:**
	 - Launch Jupyter Notebook or JupyterLab in this directory:
		 ```bash
		 jupyter notebook HierarchicalClustering.ipynb
		 ```

## Usage

Follow the notebook sequentially for:
- Theoretical background and algorithm steps
- Visual exploration of clustering and dendrograms
- Practical clustering on synthetic and Iris datasets
- Evaluation of clustering quality using silhouette scores
